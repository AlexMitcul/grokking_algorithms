# 1.1

Задание:
Отсортированный массив из 128 элементов. Какое максимальное количество проверок необходимо?

Решение:
log128 = 7. 

Ответ: 7

# 1.2

Задание:
Размер списка увеличился вдвое. Как изменится максимальное количество проверок?

Решение:
Т.к. зависимость логарифмическая, то максимальное количество проверок увеличится на одну проверку

Ответ: на одну проверку

# 1.3

Задание:
Определить время выполнение задачи: поска фамилии в справочнике.

Решение:
Обычным перебором - O(n^2)
Бинарным поиском - O(logn)

Ответ: O(n) или O(logn)

# 1.4

Задание:
Определить время выполнение задачи: Поиск фамилии в справочнике зная номер телефона человека.

Решение:
Т.к. информация в справочнике отсортирована по ФИО, то для поиска нужного номера телефона небходимо просмотреть все записи, соответственно, O(n)

Ответ: O(n)

# 1.5

Задание:
Определить время выполнение задачи: Прочитать телефоны  всех людей в телефонной книге.

Решение:
См. п. 1.4

Ответ: O(n)
